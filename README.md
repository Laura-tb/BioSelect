# Supermercado Online de Productos Ecológicos

Proyecto de **diseño de interfaz y experiencia de usuario (UI/UX)** para un supermercado online de productos ecológicos, enfocado en la claridad visual, la usabilidad y la coherencia de marca.

## Descripción del proyecto

Este proyecto aborda el diseño de una plataforma de ecommerce para la venta de productos ecológicos. El alcance incluye el diseño de las principales pantallas del flujo de compra:

- Página de inicio con categorización de productos  
- Sección de productos destacados y ofertas  
- Página de detalle de producto  

El diseño se ha planteado con el objetivo de ofrecer una experiencia de navegación intuitiva y alineada con los valores de sostenibilidad y consumo responsable.

## Objetivos del proyecto

- Diseñar una interfaz clara y orientada al usuario  
- Aplicar principios de jerarquía visual y diseño de interacción  
- Definir una identidad visual coherente con el concepto ecológico  
- Simular un caso real de ecommerce aplicable a un entorno profesional  

## Proceso de diseño

El proyecto se desarrolló siguiendo un proceso estructurado de diseño:

1. Análisis del contexto y del usuario  
2. Definición de la arquitectura de la información  
3. Creación de mockups de media fidelidad en Figma  
4. Validación de la jerarquía visual y los flujos de navegación  

Este enfoque permitió tomar decisiones fundamentadas antes de la fase de desarrollo, reduciendo iteraciones posteriores.

## Diseño de pantallas

### Página de inicio
Diseñada como punto principal de entrada al catálogo, facilita la exploración mediante:
- Acceso rápido a categorías  
- Destacado de productos clave y promociones  
- Estructura modular que favorece el escaneo visual  

### Página de detalle de producto
Centrada en apoyar la conversión y la toma de decisiones. Se prioriza:
- Imagen y precio como elementos principales  
- Llamada a la acción clara  
- Información secundaria organizada de forma jerárquica  

## Sistema visual

### Paleta de colores
La paleta se basa en tonos verdes que refuerzan la asociación con lo ecológico, la salud y la confianza. Los colores de apoyo se utilizan para estados informativos, mientras que la escala de grises permite estructurar el contenido sin generar ruido visual.

### Tipografía
Se ha seleccionado **Poppins (Google Fonts)** por su alta legibilidad, versatilidad y estética contemporánea, adecuada para interfaces digitales orientadas a ecommerce.

## Diseño y funcionamiento

El diseño sigue un enfoque **mobile-first**, garantizando una experiencia consistente en distintos dispositivos.  
La navegación es sencilla, predecible y orientada a objetivos, facilitando que el usuario encuentre productos y acceda a su información de forma eficiente.

## Aprendizajes y aplicabilidad profesional

Este proyecto ha reforzado la importancia de un proceso de diseño estructurado, donde la experiencia de usuario y la jerarquía visual guían todas las decisiones.  
Las habilidades desarrolladas son directamente aplicables a proyectos profesionales de diseño web y ecommerce, especialmente en entornos orientados a producto y negocio.
